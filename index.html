Welcome to the FASTA Histogram Creator.

FASTA Files

FASTA files contain genomic information as well as a name for that sequence. The format of the file looks like:

>name_of_read

NUCLEOTIDE_SEQUENCE

Program Requirements

In order to run the Python program drawHistogram.py, you MUST have pylab installed. The program uses Python's pylab tool to create its visualization. You can run this program on the command line, by specifying the required parameter of a FASTA file and optional parameter (num_bins) of the number of bins (default is 10). You can call the program in the command line as follows:

[user@localhost] $ drawHistogram.py fasta_file.fasta -num_bins 25

Output Histogram

This program produces a histogram of the lengths of the reads of the input FASTA. Each FASTA input can contain thousands of reads and it is often difficult to get a good understanding of your read sizes by simply looking at the FASTA. The histogram produced shows the distribution of your read lengths, with size on the x-axis and number of reads in that bin on the y-axis.
